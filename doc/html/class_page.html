render ( $  twig_renderer)

Effectue le rendus de la page.

1. Charge et execute le contenus du fichier PHP dans module associé. 2. Charge les paramètres de sécurité et fait les vérifications qui vont bien. (si l'utilisateur n'est pas autorisé, on redirige vers /login) 3. Lance Twig.

Dans les modules les variables suivantes sont définies :

  • $Page : Une référence vers cet objet
  • $Securite : Un accés directe à la sécurité
